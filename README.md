# AhorroYA (Expo + React Native)

Proyecto base de Expo para AhorroYA con la estructura solicitada:

- `App.js`
- `screens/ResultsScreen.js`
- `components/ActivityFeed.js`
- `app.json`
- `package.json`

## Estructura

```text
.
├── App.js
├── app.json
├── components/
│   └── ActivityFeed.js
├── package.json
└── screens/
    └── ResultsScreen.js
```

## Ejecución exacta

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar Expo:

```bash
npm run start
```

3. (Opcional) Ejecutar en plataforma específica:

```bash
npm run android
npm run ios
npm run web
```
