# AhorroYA Valuation Scenarios

Date: 2026-05-11

## Disclaimer

This is an internal strategic valuation framework, not a professional appraisal, fairness opinion, tax valuation, or investment recommendation. No real traction or revenue is assumed unless separately evidenced.

## Valuation Components

Potential value sources:

- Technology built.
- Product/growth opportunity.
- Future proprietary price/search data.
- Strategic value to retailers, brands, coupon platforms, fintechs, or marketplaces.

Current valuation discounts:

- Production not live.
- PayPal live not enabled.
- Google OAuth production not verified.
- Real user traction not yet proven.
- Data coverage not yet proven at scale.

## Scenario 1: Conservative

Assumptions:

- Staging product remains pre-revenue.
- First 100 users produce mixed feedback.
- Data coverage requires manual work.
- Premium conversion remains unproven.

Indicative value logic:

- Mainly technology asset value plus early product option value.
- Buyer would discount heavily for execution risk.

Orientative range:

- USD 25k-100k equivalent strategic/project value.

## Scenario 2: Base

Assumptions:

- First 100 users validate search-to-result activation.
- 10-25 users share savings or create retention actions.
- PayPal live and OAuth production gates close.
- Early Premium or partner interest appears.

Indicative value logic:

- Early pre-seed opportunity with working MVP and validated user pain.
- Value increases if data ingestion and retention become repeatable.

Orientative range:

- USD 150k-500k pre-seed-style valuation discussion.

## Scenario 3: Aggressive

Assumptions:

- Uruguay launch shows strong retention.
- Premium conversion is measurable.
- Retail/brand partnership interest is real.
- Data coverage expands beyond Montevideo.
- LATAM playbook becomes credible.

Indicative value logic:

- Strategic consumer-data/commerce platform option.
- Potential buyers include retail tech, coupon/loyalty, marketplace, fintech, or CPG data buyers.

Orientative range:

- USD 750k-2.5M+ strategic/pre-seed-to-seed discussion.

## Value By Technology Built

Current built assets:

- App frontend and user flows.
- API/backend.
- Supabase/RLS model.
- PayPal sandbox integration.
- Tracking layer.
- AI agent safe architecture.
- Release and security documentation.

Technology value is meaningful but not enough alone to justify high valuation without user/data validation.

## Value By Market Opportunity

Market upside depends on:

- Grocery spend frequency.
- Inflation sensitivity.
- WhatsApp acquisition.
- Retail price dispersion.
- Ability to build reliable data partnerships.

## Value By Future Data

Possible future data assets:

- Search demand by product/category.
- Price spread by store/region.
- Savings behavior.
- Alert/favorite intent.
- Product availability signals.

Current status: future potential, not yet defensible as a mature dataset.

## Strategic Buyer Value

Possible strategic buyers/partners:

- Supermarket chains.
- Coupon/loyalty platforms.
- CPG brands.
- Fintechs with consumer savings positioning.
- Local commerce marketplaces.
- Data/market intelligence firms.

Strategic value increases if AhorroYA demonstrates user demand and trusted price coverage.

## Risks That Discount Valuation

- No production launch yet.
- Payment live not verified.
- User retention unknown.
- Data freshness risk.
- Trust/compliance risk if price claims are too strong.
- Competitive response from incumbents.
