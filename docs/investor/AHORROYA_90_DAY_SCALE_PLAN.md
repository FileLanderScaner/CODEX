# AhorroYA 90 Day Scale Plan

Date: 2026-05-11

## Goal

Move from staging-ready product to validated first users, production readiness, and an investor/socios package grounded in evidence.

## Days 1-15: Close Production Blockers

Objectives:

- Complete PayPal live setup.
- Verify Google OAuth production.
- Verify Supabase Auth leaked password protection.
- Produce real backup/revert evidence.
- Keep staging available for demos.

KPIs:

- Production gate blocker count reduced to zero or documented external owner/date.
- PayPal live test completed safely.
- OAuth production redirect verified.
- Backup artifact created and restore/revert plan validated.

Success:

- `PASS_PRODUCTION_READY` becomes possible.

Failure:

- Any live credentials remain unavailable.
- Any dashboard setting cannot be verified.

Suggested owners:

- DevOps/Release Manager.
- Security Engineer.
- Payments Engineer.

## Days 16-30: First 100 Users

Objectives:

- Invite first controlled testers.
- Validate search-to-savings activation.
- Collect missing product feedback.
- Measure WhatsApp loop.

KPIs:

- 100 app opens.
- 50 searches.
- 25 cheapest-price views.
- 10 WhatsApp shares or copied savings.
- 5 Premium CTA clicks.

Success:

- Users understand the value in under 5 seconds.
- Users get savings evidence in under 30 seconds.

Failure:

- Searches return empty too often.
- Users do not trust price data.
- No sharing occurs.

Suggested owners:

- Growth Lead.
- Product Manager.
- Data Analyst.

## Days 31-60: Retention, Savings, Monetization Validation

Objectives:

- Improve data coverage for top searched products.
- Validate favorites and alerts retention.
- Run Premium willingness-to-pay interviews.
- Keep PayPal live controlled if production gate passes.

KPIs:

- 20 repeat users.
- 30 favorites/alerts.
- 10 user interviews.
- 1-3 successful controlled payment tests if live gates pass.

Success:

- Users return to check repeated products.
- Premium value is understood.

Failure:

- No repeat behavior.
- Premium CTA does not convert to interest.

Suggested owners:

- Product Manager.
- Monetization Lead.
- Full-Stack Engineer.

## Days 61-90: Partnerships, Growth Loops, Seed/Pre-Seed Prep

Objectives:

- Approach small retailers/brands.
- Build first B2B insight sample.
- Prepare seed/pre-seed narrative from actual metrics.
- Tighten pitch deck and due diligence.

KPIs:

- 3-5 partnership conversations.
- 1 B2B sample report.
- 1 updated investor memo with real user metrics.
- First repeatable acquisition channel identified.

Success:

- Evidence supports investor-ready package.

Failure:

- No partner interest.
- Growth depends only on manual founder effort.

Suggested owners:

- CEO/Growth.
- Business Development.
- Data Analyst.
