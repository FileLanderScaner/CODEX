# AhorroYA User Interview Script

## Opening

"Gracias por probar AhorroYA. Esto es una version staging protegida para validar si la app ayuda a comparar precios y encontrar ahorro. No es produccion publica y PayPal live todavia no esta activo."

## Warmup

- How do you currently compare grocery prices?
- How often do you choose a store based on price?
- What is the most annoying part of saving on groceries?

## Task 1: Understand The App

Ask the user to look at the first screen for 5 seconds.

Questions:

- What do you think this app does?
- What would you do next?
- Is anything unclear?

## Task 2: Search

Ask the user to search for a common grocery product.

Observe:

- Did they find the search input?
- Did they understand suggestions/results?
- Did they need help?
- Did any UI element confuse them?

## Task 3: Compare And Save

Ask the user to explain the cheapest result.

Questions:

- Which store would you choose?
- How much do you think you save?
- Do you trust this result?
- What extra information would help?

## Task 4: Share

Ask the user to share or copy the savings result.

Questions:

- Would you send this to someone?
- Is the WhatsApp message natural?
- Would this make you invite another user?

## Task 5: Premium

Show the Premium CTA.

Questions:

- What do you think Premium gives you?
- Which benefit is most valuable?
- What would make you pay?
- What should remain free?

## Close

- Would you use AhorroYA again this week?
- Who else should test this?
- What is the most important thing to improve before public launch?
