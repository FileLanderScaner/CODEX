# AhorroYA WhatsApp Outreach Script

## Operator Rules

- Start with 10 testers, not 100.
- Do not post the staging link in large public groups.
- Do not share bypass tokens in messages.
- Do not describe the app as production-ready.
- Do not ask users to enter sensitive data.
- Record every response in the feedback triage board.

## One-To-One Sequence

### Message 1: Invitation

Hola, estoy armando una prueba controlada de AhorroYA, una app para comparar precios de supermercado y encontrar ahorro estimado.

Todavia no es produccion publica. Es staging protegido, puede tener errores y no hay pagos reales activos. No ingreses datos sensibles innecesarios.

Te puedo mandar el link para que pruebes una busqueda y me des feedback en 5 minutos?

### Message 2: Link Delivery

Gracias. Proba esto:

1. Abri el link.
2. Busca un producto comun.
3. Mira cual aparece mas barato.
4. Intenta compartir o copiar el ahorro.
5. Respondeme que fue claro y que te genero duda.

Recordatorio: es una prueba staging, no produccion publica.

### Message 3: Follow-Up After 30 Minutes

Pudiste probar AhorroYA?

Me sirve responder estas 3 cosas:

1. Entendiste en segundos para que sirve?
2. Encontraste algun ahorro o comparacion util?
3. Que deberiamos mejorar antes de mostrarla a mas gente?

### Message 4: 48-Hour Follow-Up

Gracias por probar. Dos preguntas finales:

1. Volverias a usar AhorroYA antes de comprar en el super?
2. Que funcion te haria pagar Premium: alertas, historial de precios, carrito inteligente, favoritos ilimitados u otra?

## Group Post Sequence

Post only in trusted small groups:

Estoy validando AhorroYA con un grupo chico. Es una app para comparar precios de supermercado y ver ahorro estimado.

No es produccion publica todavia. Es staging, puede tener errores y no hay pagos reales activos.

Si queres ayudar, mandame privado y te paso la prueba. Busco feedback honesto de 5 minutos.
