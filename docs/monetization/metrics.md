# Monetization metrics

Metricas objetivo:

- `premium_conversion_rate`: suscripciones activas / usuarios que vieron paywall.
- `churn`: cancelaciones / suscripciones activas al inicio del periodo.
- `affiliate_ctr`: clicks afiliados / impresiones.
- `affiliate_revenue`: ingresos atribuidos por `signed_ref`.
- `mrr`: suma de suscripciones mensuales normalizadas.
- `aru` / `arpu`: ingresos por usuario activo / usuario total.
- `source_freshness_sla`: fuentes actualizadas dentro de ventana esperada.
- `import_success_rate`: jobs exitosos / jobs totales.
