# AhorroYA Demo Script For First 100 Users

## Opening

"AhorroYA te muestra dónde comprar más barato en segundos. Esta demo está en staging protegido, no en producción pública. PayPal live todavía no está activo; cualquier flujo de pago mostrado es sandbox o explicativo."

## Demo Flow

1. Open protected staging.
2. Show the headline and value proposition.
3. Search for a common product.
4. Show price comparison results.
5. Point out the cheapest available option.
6. Explain estimated savings without promising guaranteed savings.
7. Open a product detail.
8. Show how the user can share the savings by WhatsApp.
9. Use copy fallback if WhatsApp is not available.
10. Show the Premium CTA.
11. Explain Premium as the intended monetization path.
12. Clarify that PayPal live is blocked until live credentials and webhook evidence are complete.

## Reviewer Questions During Demo

- Did the value become clear in the first 5 seconds?
- Could you find a product without explanation?
- Was the estimated saving credible?
- Would you share this with someone who compares supermarket prices?
- What would make you trust the prices more?
- Which Premium feature would make you pay?

## Safe Claims

- "Compará precios disponibles."
- "Encontrá oportunidades de ahorro."
- "Ahorro estimado."
- "Staging protegido para validación."
- "PayPal live pendiente de credenciales y evidencia."

## Claims To Avoid

- "Producción lista."
- "Ahorro garantizado."
- "Siempre el precio más bajo."
- "Pagos reales activos."
- "Datos en tiempo real" unless the data source is verified for that claim.

## Close

"La meta de esta revisión es validar si usuarios reales entienden el valor, encuentran ahorro y compartirían AhorroYA. Producción pública sigue bloqueada hasta cerrar credenciales y evidencia externa."
