# Metrics plan

Eventos core:

- `landing_view`
- `open_app`
- `app_loaded`
- `web_session_started`
- `client_error`
- `search_submitted`
- `search_product`
- `view_best_price`
- `cheapest_price_shown`
- `fallback_used`
- `commerce_clicked`
- `share`
- `share_click`
- `click_whatsapp`
- `add_favorite`
- `create_alert`
- `premium_click`
- `premium_started`
- `premium_completed`

Funnels:

- Busqueda a resultado.
- Resultado a WhatsApp/share.
- Busqueda a favorito.
- Favorito a alerta.
- Premium click a pago.

Dashboards recomendados:

- Productos mas buscados.
- Tiendas mas vistas.
- Ahorro total mostrado.
- Tiempo a primer resultado.
- Conversion free a premium.
- Errores API por ruta.
- Fallbacks por comercio.
- Errores cliente agregados sin datos personales.
